# KNN and Radius Neighbors Classification – Wine Dataset

This project evaluates the performance of two distance-based machine learning classifiers:

- K-Nearest Neighbors (KNN)
- Radius Neighbors (RNN)

The experiment uses the Wine dataset from the sklearn library to analyze how parameter selection affects classification accuracy.

## Objectives

- Implement KNN using multiple k values
- Implement RNN using different radius values
- Compare classification accuracy
- Visualize performance trends
- Analyze when each model performs best

## Resources Used

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
